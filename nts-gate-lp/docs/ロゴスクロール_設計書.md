# ロゴスクロールセクション 設計書

## 1. 概要

**配置:** Hero セクション（S1）の下端、ピルタグとスクロール chevron の間  
**目的:** 提携先企業のロゴを無限スクロールで表示し、社会的証明（Social Proof）を提供  
**トーン:** 控えめ。ロゴの存在でさりげなく信頼感を補強する。メインコピーを邪魔しない。

## 3. デザイン仕様

### 3-1. セクション全体

| 項目 | 値 |
|------|-----|
| 高さ | auto（ロゴ高さ + 上下パディング） |
| 上マージン | 48px（mt-12） |
| 下マージン | 0（chevron までの余白で調整） |
| 背景 | transparent（Hero の背景と一体化） |
| 左右オーバーフロー | hidden（スクロールがはみ出さない） |

### 3-2. ラベルテキスト（任意）

| 項目 | 値 |
|------|-----|
| テキスト | 「提携企業」or なし（ロゴのみ） |
| フォント | var(--text-caption), font-weight: 400 |
| 色 | var(--neutral-500) |
| 配置 | 中央寄せ、ロゴ列の上 |
| 表示/非表示 | デスクトップのみ表示、モバイルでは非表示にしても可 |

### 3-3. ロゴ列

| 項目 | 値 |
|------|-----|
| ロゴ高さ | 24px（デスクトップ）/ 20px（モバイル） |
| ロゴ幅 | auto（アスペクト比維持） |
| ロゴ間の余白 | 48px（gap-12）デスクトップ / 32px モバイル |
| ロゴの色 | グレースケール、opacity: 0.4 → ホバーで 0.7 |
| フィルター | `grayscale(100%) opacity(0.4)` |
| ホバー | `grayscale(0%) opacity(0.7)` transition 300ms |

### 3-4. スクロールアニメーション

| 項目 | 値 |
|------|-----|
| 方向 | 左 → 右 に流れる（marquee 方式） |
| 速度 | 30秒で全ロゴ1周（--scroll-duration: 30s） |
| 停止条件 | ホバー時に一時停止 |
| 実装方式 | CSS @keyframes + translateX(-50%) |
| 無限ループ | ロゴ配列を2回繰り返して配置し、50%移動で巻き戻す |
| GPU最適化 | transform のみ使用、will-change: transform |

```css
@keyframes logo-scroll {
  0%   { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}
```

| Desktop (> 1024px) | 24px | 48px | 30s |

---

## 5. アクセシビリティ

- `aria-hidden="true"` — ロゴスクロールはデコレーション要素
- `prefers-reduced-motion: reduce` — アニメーション停止、静的にロゴを並べる
- ロゴ画像に alt テキストは「[企業名] ロゴ」を設定（スクリーンリーダーからは aria-hidden で非表示）
- 十分なコントラスト（グレースケール + 薄い opacity は装飾目的のため可）

---

## 6. パフォーマンス

- ロゴは SVG or WebP（軽量）
- 画像は `/public/logos/` に配置（Next.js の static file serving）
- LCP に影響しないよう、ロゴ画像は `loading="lazy"` は不要（viewport 内だが小さい装飾）
- CSS アニメーションのみ（JS 不使用 = Framer Motion 不使用）

## 8. コンポーネント Props

```typescript
interface LogoItem {
  src: string;     // /logos/company-a.svg
  alt: string;     // "企業A ロゴ"
  width?: number;  // 任意の幅指定（アスペクト比維持用）
}

interface LogoScrollProps {
  logos: LogoItem[];
  speed?: number;           // アニメーション速度（秒）デフォルト 30
  className?: string;       // 追加の Tailwind クラス
  label?: string;           // ラベルテキスト（「提携企業」等）
}
```

## 10. ロゴ素材の準備

### 現状
- `assets/NTS.logo.svg` のみ存在
- 提携先企業のロゴファイルは未提供

### 必要なアクション
1. 提携先企業のロゴ画像（SVG 推奨）を `public/logos/` に配置
2. ロゴは高さ 24-32px 程度で表示されるため、過度に詳細な画像は不要
3. 色付きロゴでもグレースケールフィルターで統一表示される
4. **提供されるまではプレースホルダー（企業名テキスト）で仮実装する**

### ロゴ追加手順
```typescript
// LogoScroll 呼び出し時にロゴ配列を更新するだけ
const logos = [
  { src: "/logos/company-a.svg", alt: "企業A ロゴ" },
  { src: "/logos/company-b.svg", alt: "企業B ロゴ" },
  // ... 追加
];
```

## 11. 心理学的根拠

- **社会的証明（Social Proof）**: 他の企業がすでに提携している事実が、
  新規ユーザーの信頼を構築する
- **バンドワゴン効果**: 「多くの企業が使っている」印象を与える
- **控えめな配置**: ロゴは薄くグレーで表示することで、
  押し売り感を排除しつつ信頼の下地を築く
